I've noticed [Visual Studio Code](https://code.visualstudio.com/) in a few videos, and admired what a clean interface it had, and was impressed how opening a terminal window was automatically in the directory you were working in.

I had a need to write some html/css, and some C++ in the last couple of days, so that seemed like a great excuse to give it a try. I'd have to say my opinion of it has only gone up. Clearly, it is right at home with HTML and CSS - code completion and syntax colouring all working nicely. I followed TechWithTim's suggestion to install the [Live Server extension](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) - which was a completely painless experience.

Same positive experience when I created a .cpp file, and VSC wanted me to take Microsoft's advice about a bunch of extensions, and correctly suggested I build with Mac's clang compiler.

![](/images/screen-shot-2022-08-11-at-3.40.15-pm.jpg)

So many things just work how you expect them to. Within seconds I was trying out breakpoints, compiling and managing the git commits. I even tried the Swift extension.

I have no intention of leaving Xcode, but with a great tool like this, I will never be a [vim](/vim/) guy.
